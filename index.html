Proyectos

>Discord

A modo de práctica intenté replicar la plataforma Discord usando PHP y MySQL...

Implementé un sistema de registro e inicio de sesión, solicitudes de amistad, creación de servidores y chat en vivo con encriptación

Próximamente quiero volver a este proyecto hasta implementar todas las características de Disord menos el chat por voz.

Lo que más me gustó del desarollo fue diseñar una "plantilla de servidor" y aprender como mediante PHP modificar copias de esa plantilla para crear nuevos serividores según las especificaciones del usuario.

También fue mi primera vez usando AJAX.

>Página Arcade

Mi proyecto más ambicioso hasta la fecha ha sido la creación una página web para una empresa de máquinas arcade...

Por toda la página escondí lugares donde cuando el usuario interactuaba con ellos conseguía una moneda se añadía a un marcador global usando Node.js y WebSockets.

Repliqué el juego "Pacman" usando la etiqueta canvas. También implementé un ranking con WebSockets

Toda la página tiene temática de juegos retro y practicamente todo es interactuable. Mi parte favorita es una sección con información de la empresa incrustada en planetas. Cuando el usuario hace click en uno de ellos una nave dispara que hace volar el planeta en mil pedazos

También creé diseños propios para la sección de precios, usando marquesinas icónicas de arcades con Photoshop.

>Spotify API

Por último, algunos proyectos de menor envergadura usando React que también me ayudaron a crecer como desarollador...

Mi primera web con React fue un buscador de artistas que implementaba la API de Spotify.

Me ayudó a entender la programación asíncrona, sobre fetch y sobre enviar solicitudes a APIs en general.

También cree un pequeño "minijuego" para practicar puntería que imitaba un juego de tipo shooter usando Three.js

Me ayudó a entender los principios de Three.js y sobre implementar elementos 3D a una pagina web. Además aprendí cosas relacionadas con los videojuegos como los raycasters
